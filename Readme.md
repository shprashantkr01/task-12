********* Working ********
The script.js contains a function named evenOrOdd that takes a number as input and returns a string indicating whether the number is even or odd by simply performing the modulo operation.
if number%2 == 0 this indicated the number is divisible by two meaning number is even.
if number%2 != 0 this indicated the number is not divisible by two meaning number is odd.
