>>>>>>>>> How to Run in Browser.
step 1: clone the repository in you machine.
step 2: add live server extension in vs code.
step 3: open project folder.
step 4: click go live in the bottom right corner of the vs code window. 
step 5: go to browser and open inspect window.
step 6: open console to see the result.

This project successfully fetches the data from an array and displays it in card format on the screen.
The search bar is fully functional with the help of filter method in arrays.
